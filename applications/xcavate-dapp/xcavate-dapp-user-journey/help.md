# Help

Click on the ? Help icon and either select a pre-formatted question or type a question in the prompt area clicking the blue Message button to initiate the query.

<figure><img src="../../../.gitbook/assets/3.1.0 Help (1).jpg" alt="" width="375"><figcaption></figcaption></figure>

If you do not get a sufficient response, then try adjusting your prompt question. If you have a technical problem or you do not get a satisfcatory response, then please type 'create a support ticket'. You will need to type in some details about the issue, which will be logged and a response sent to your account message inbox asap.
