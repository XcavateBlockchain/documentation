# Installation (iOS & Android)

1.  iOS

    1. Use the following testing invitation link. [https://testflight.apple.com/join/2211G9eW](https://testflight.apple.com/join/2211G9eW)
    2. Install TestFlight (if not already having this app). This is a free Apple app in the App Store.
    3. Testers will be added to the testers list.


2. Android
   1. Use the following testing invitation link. [https://play.google.com/apps/testing/com.xcavate.realxmarket](https://play.google.com/apps/testing/com.xcavate.realxmarket)
   2. Testers will be added to the testers list.
